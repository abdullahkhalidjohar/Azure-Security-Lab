# ☁️ Azure Cloud & Security Lab

### 📋 Overview
A hands-on implementation of a cloud infrastructure environment on Microsoft Azure, focused on automated monitoring and security operations.

### 🛠️ Technical Stack
* **Cloud Platform:** Azure Infrastructure (Virtual Machines, VNet, NSG, Key Vault).
* **Security & Monitoring:** Log Analytics Workspace, Azure Monitor Agent (AMA), Data Collection Rules (DCR).
* **Skills:** Infrastructure as a Service (IaaS) deployment, security telemetry, and log management.

### 🔍 Project Highlights
* Designed a multi-resource Azure environment to simulate enterprise cloud setups.
* Configured centralized logging to collect Windows security events.
* Implemented Azure Monitor Agent (AMA) for efficient telemetry collection.
